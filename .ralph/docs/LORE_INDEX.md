# Emberveil Lore Index

> This index maps worldbuilding topics to their source documents in Google Docs
> and their corresponding project specs. Ralph should consult this file first
> when needing lore context for any development task.

## How to Use This Index
1. Find the topic relevant to your current task
2. Read the related spec in `.ralph/specs/` for implementation details
3. If deeper lore context is needed, fetch the Google Doc via its link
4. Google Docs are the single source of truth - specs derive from them
